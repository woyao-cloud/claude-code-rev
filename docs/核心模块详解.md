## 1. 初始化流程

**文件**: `src/entrypoints/init.ts`

### 1.1 初始化函数 `init()`

采用 `memoize` 记忆化，确保只执行一次：

```typescript
export const init = memoize(async (): Promise<void> => {
  const initStartTime = Date.now()
  
  // 1. 配置系统启用
  enableConfigs()
  
  // 2. 应用安全环境变量 (信任对话框之前)
  applySafeConfigEnvironmentVariables()
  
  // 3. 应用 CA 证书配置 (TLS 握手之前)
  applyExtraCACertsFromConfig()
  
  // 4. 设置优雅关闭
  setupGracefulShutdown()
  
  // 5. 初始化遥测 (延迟加载 ~400KB OpenTelemetry)
  void Promise.all([
    import('../services/analytics/firstPartyEventLogger.js'),
    import('../services/analytics/growthbook.js'),
  ]).then(([fp, gb]) => {
    fp.initialize1PEventLogging()
    gb.refreshGrowthBook()
  })
  
  // 6. 策略限制初始化
  if (isPolicyLimitsEligible()) {
    initializePolicyLimitsLoadingPromise()
  }
  
  // 7. 远程设置初始化
  if (isEligibleForRemoteManagedSettings()) {
    initializeRemoteManagedSettingsLoadingPromise()
  }
  
  // 8. OAuth 账户信息填充
  populateOAuthAccountInfoIfNeeded()
  
  // 9. LSP 服务器管理器关闭注册
  registerCleanup(async () => {
    await shutdownLspServerManager()
  })
})
```

```typescript
export async function findGitRoot(cwd: string): Promise<string | null> {
  let current = cwd
  while (current !== path.dirname(current)) {
    if (existsSync(path.join(current, '.git'))) {
      return current
    }
    current = path.dirname(current)
  }
  return null
}
```

### 7.1 LSP 管理器

```typescript
// manager.ts
export class LSPServerManager {
  private static instance: LSPServerManager
  private servers: Map<string, LSPServerInstance>
  
  static getInstance(): LSPServerManager {
    if (!this.instance) {
      this.instance = new LSPServerManager()
    }
    return this.instance
  }
  
  async initializeServer(config: LSPConfig): Promise<string> {
    const server = new LSPServerInstance(config)
    await server.initialize()
    
    const serverId = generateServerId(config)
    this.servers.set(serverId, server)
    
    return serverId
  }
  
  async shutdown(): Promise<void> {
    for (const server of this.servers.values()) {
      await server.shutdown()
    }
    this.servers.clear()
  }
}
```

```typescript
export class LSPClient {
  private connection: Connection
```

```typescript
// grouping.ts
export function groupMessagesByTopic(
  messages: Message[]
): MessageGroup[] {
  const groups: MessageGroup[] = []
  let currentGroup: Message[] = []
  
  for (const message of messages) {
    if (shouldStartNewGroup(message, currentGroup)) {
      if (currentGroup.length > 0) {
        groups.push({ messages: currentGroup })
      }
      currentGroup = [message]
    } else {
      currentGroup.push(message)
    }
  }
  
  if (currentGroup.length > 0) {
    groups.push({ messages: currentGroup })
  }
  
  return groups
}
```

## 10. 总结

### 核心架构模式

1. **懒加载**: 大量使用动态导入减少启动时间
2. **记忆化**: 使用 `lodash-es/memoize` 缓存昂贵操作
3. **特征标志**: 使用 GrowthBook 进行功能控制
4. **服务分层**: 清晰的服务层划分，职责分离
5. **类型安全**: TypeScript 优先，完整的类型定义

### 关键设计决策

1. **快速路径启动**: 特殊标志零模块加载
2. **遥测延迟加载**: 避免启动时加载 ~400KB OpenTelemetry
3. **条件性工具**: 基于 feature flag 启用/禁用工具
4. **权限分级**: 多级权限模式，适应不同场景
5. **MCP 可扩展**: 支持外部 MCP 服务器集成
