   核心模块详解

  1. 启动流程模块 (Bootstrap)

  文件: src/bootstrap-entry.ts → src/main.tsx

  启动流程采用快速路径 (fast-path) 设计：

  // bootstrap-entry.ts
  ensureBootstrapMacro()
  await import('./entrypoints/cli.tsx')

  快速路径优化 (在 main.tsx 中处理):
  - --version - 零模块加载
  - --daemon - 守护进程模式
  - --remote-control - 远程控制模式
  - --background - 后台模式

  关键初始化步骤:
  1. 启动性能分析 (profileCheckpoint)
  2. MDM 设置预读 (startMdmRawRead)
  3. Keychain 预取 (startKeychainPrefetch)
  4. 特征标志加载 (feature())
  5. 环境变量应用
  6. 遥测初始化
  7. 策略限制加载

  命令类型

  type Command =
    | { type: 'prompt' }      // 模型可调用
    | { type: 'local' }       // 本地执行
    | { type: 'local-jsx' }   // 终端 UI 渲染

  动态加载机制

  // 条件性加载 (feature flag 控制)
  const assistantCommand = feature('KAIROS')
    ? require('./commands/assistant/index.js').default
    : null
